# Project Brief

## Project Name
KikiLuna Beauty Mobile App

## Goal
Build a Flutter mobile shopping app for kikilunabeauty.com powered by Shopify.

## Target Users
Women in the USA and Canada, especially Arabic-speaking and English-speaking customers interested in beauty, self-care, perfumes, oils, and Moroccan products.

## Business Value
Provide a smoother mobile shopping experience than the website, improve retention, increase repeat purchases, and support push notifications and a branded experience.

## MVP Scope
- Splash screen
- Home screen
- Collections
- Product details
- Search
- Cart
- Customer login/signup
- Shopify checkout handoff
- Arabic/English support

## Non-Goals
- Full custom backend
- In-app checkout processing
- Complex loyalty program in MVP
- Advanced AI features in MVP
