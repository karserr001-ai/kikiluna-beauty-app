# Product Requirements

## Product Name
KikiLuna Beauty Mobile App

## Overview
A Flutter mobile shopping app for Kikiluna Beauty powered by Shopify, designed to give customers a smoother, more branded mobile shopping experience than the website.

## Objectives
- Increase mobile conversions
- Improve repeat purchases
- Create a better shopping experience for beauty customers
- Support bilingual shopping (Arabic and English)
- Make browsing, cart, and checkout easy on mobile

## Target Users
- Women in the USA and Canada
- Arabic-speaking customers
- English-speaking customers
- Customers interested in beauty, skincare, perfumes, oils, and Moroccan products

## MVP Features
- Splash screen
- Onboarding or welcome flow
- Home screen
- Collections list
- Collection products screen
- Product details screen
- Search
- Cart
- Customer login / signup
- Profile / account
- Checkout handoff to Shopify
- Arabic / English language support

## Post-MVP Features
- Wishlist
- Push notifications
- Order tracking
- Reviews
- Loyalty / rewards
- Personalized recommendations
- Discount banners and campaign sections
- Social sharing

## Core User Stories
- As a user, I want to browse collections so I can discover products easily.
- As a user, I want to search for products so I can find what I need quickly.
- As a user, I want to view product details so I can understand ingredients, size, and benefits.
- As a user, I want to add products to cart so I can buy multiple items.
- As a user, I want to log into my account so I can view my orders and saved details.
- As a user, I want to switch language so I can shop comfortably in Arabic or English.
- As a user, I want to go to checkout smoothly so I can complete my purchase.

## Screen List
- Splash
- Welcome / onboarding
- Home
- Collections
- Collection details
- Product details
- Search
- Cart
- Login
- Signup
- Forgot password
- Account
- Orders
- Settings

## Functional Requirements
- The app must fetch products and collections from Shopify.
- The app must display product images, titles, prices, descriptions, and variants.
- The app must allow add to cart, remove from cart, and quantity changes.
- The app must support customer authentication.
- The app must support checkout handoff to Shopify.
- The app must support Arabic and English.
- The app must handle loading, empty, and error states.

## Non-Functional Requirements
- Fast loading
- Responsive UI
- Clean and premium design
- Stable navigation
- Secure API usage
- Easy maintainability
- Scalable architecture

## Success Metrics
- App launch readiness
- Smooth browsing and cart flow
- Low crash rate
- Good mobile performance
- Increased repeat customer engagement

## Risks / Constraints
- Shopify limitations for mobile checkout
- API rate limits
- Time required for bilingual content
- Keeping the app design consistent with the brand
